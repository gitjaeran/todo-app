### ✨Start Project
```
git clone [url]
cd server/
npm i
npm start # PORT NUM: 8080
cd ../client
npm i
npm start # PORT NUM: 3000
```
<br/>

### 📌REST API
`GET /todos` - show all todos  
`POST /todo` - create a new todo  
`PATCH /todo/:todoId` - edit a specific todo  
`DELETE /todo/:todoId` - remove a specific todo
